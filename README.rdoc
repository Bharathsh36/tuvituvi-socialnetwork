== Tuvi-Tuvi

Tuvituvi is a social networking site which helps you not only connect with your friends but also share dreams on work on that together.
(It was built when I was 15, hahaha Showing off..)

Stack:
* Ruby 2.3.3
* Rails 4.2.3
* mysql2

This app uses passenger as its backend server, paperclip for image uploading and devise for authentication.
